WHAT IS FERNOSA?

Fernosa is an open-source CSS library. It is customizable to the brink, allowing you to change the colours according to your colour theme, and the components will be generated accordingly.

You can also use the utility classes we provide to change the components to what you find suitable. It comes packed with beautiful and completely responsive components accordingly effects you can use in any project.
